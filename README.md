# gr-csitool
gr-csitool is a customized gnuradio module for sniffering 802.11 g packets and reporting CSI. It is compatible to gnuradio V3.7.

To build gr-csitool:

~~~
  mkdir build
  cd build
  cmake ..
  make
  sudo make install
  sudo ldconfig
~~~

Examples (Wi-Fi receiver) are provided in the project.
